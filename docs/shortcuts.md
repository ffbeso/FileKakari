# キーボードショートカット

テキスト入力中は、コピーや貼り付けなどの標準編集操作が優先されます。

## ナビゲーションとタブ

| キー | 操作 |
| --- | --- |
| Ctrl+L / Alt+D | パス入力へ移動 |
| Ctrl+F | 絞り込み入力へ移動 |
| Alt+Left / Alt+Right | 戻る / 進む |
| Alt+Up / Backspace | 親フォルダへ移動 |
| F5 | 更新 |
| Ctrl+T | 新しいタブ |
| Ctrl+Shift+T | 閉じたタブを復元 |
| Ctrl+W | 現在のタブを閉じる |
| Ctrl+Tab / Ctrl+Shift+Tab | 次 / 前のタブ |
| Ctrl+1～Ctrl+9 | 指定位置のメインタブへ移動 |

## ファイル一覧

| キー | 操作 |
| --- | --- |
| Enter | 選択項目を開く |
| Ctrl+Enter | 選択フォルダを新しいタブで開く |
| F2 | 名前変更 |
| F3 | プレビュー表示切替 |
| Delete | ゴミ箱へ移動 |
| Ctrl+C / Ctrl+X / Ctrl+V | コピー / 切り取り / 貼り付け |
| Ctrl+Z | 直前のファイル操作を元に戻す |
| Ctrl+A | 表示中の項目をすべて選択 |
| Ctrl+N | テキストファイルを作成 |
| Ctrl+Shift+N | フォルダを作成 |

Escは絞り込みを解除します。プレビューにフォーカスがある場合はプレビューを閉じます。

Shift+Deleteによる完全削除には対応していません。
